### Things I've built
 
**[MiniBarcelona3D](https://github.com/FabianUB/minibarcelona3d)**  — A real-time 3D visualization of Barcelona's entire public transport network. Trains move on the map as they do in real life. React + Three.js + Mapbox on the front, Go + SQLite on the back. Live at **[minibarcelona3d.com](https://minibarcelona3d.com)**
 
**[Scriber](https://github.com/FabianUB/scriber)** — A component-based PDF creator for Python, like Streamlit, but for PDFs. Drop in your pandas DataFrames and matplotlib figures and get a polished document out the other end. `pip install scriber-pdf`
 
**[Tether](https://github.com/FabianUB/tether)** — A template for building AI/ML desktop apps with Tauri + React + FastAPI. Frontend devs write React, ML devs write Python while Rust stays invisible.
